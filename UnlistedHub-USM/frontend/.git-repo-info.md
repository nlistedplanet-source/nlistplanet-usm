# Git Repository Configuration

**IMPORTANT:** This frontend MUST push to the following repository:

```
git@github.com:nlistedplanet-source/nlistplanet-usm.git
```

## Vercel Configuration
- **Vercel Project:** nlistplanet-app
- **Connected Repo:** nlistplanet-usm.git
- **Branch:** main
- **Root Directory:** . (empty/root)

## To verify correct remote:
```bash
git remote -v
# Should show: nlistplanet-usm.git
```

## If remote is wrong, fix it:
```bash
git remote set-url origin git@github.com:nlistedplanet-source/nlistplanet-usm.git
```

**DO NOT** push to `nlistplanet-frontend.git` - that repo is not connected to Vercel!
